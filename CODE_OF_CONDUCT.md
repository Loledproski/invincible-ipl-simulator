# Code of Conduct

# 🤝 Our Pledge

We, as contributors and maintainers of **Invincible – IPL Strategy Simulator**, are committed to making participation in this project a welcoming, respectful, and harassment-free experience for everyone, regardless of age, experience level, nationality, gender, personal background, or identity.

We believe that collaboration, constructive feedback, and mutual respect help create better software and a stronger open-source community.

---

# 🌟 Our Standards

Examples of behavior that contributes to a positive environment include:

- Being respectful and professional.
- Giving constructive feedback.
- Being welcoming to new contributors.
- Respecting different viewpoints and experiences.
- Helping others learn and improve.
- Focusing on what is best for the project and community.
- Communicating politely during discussions and code reviews.

Examples of unacceptable behavior include:

- Harassment or personal attacks.
- Offensive, discriminatory, or abusive language.
- Trolling or intentionally disruptive behavior.
- Publishing another person's private information without permission.
- Spam, malicious code, or intentionally harmful contributions.
- Any conduct that could reasonably be considered inappropriate in a professional environment.

---

# 🛠 Contribution Guidelines

When contributing to this project:

- Write clean and readable code.
- Follow the existing project structure.
- Test changes before submitting them.
- Keep pull requests focused on a single feature or fix.
- Clearly describe the purpose of your contribution.
- Respect maintainers' decisions during code review.

---

# 🚨 Reporting Issues

If you experience or witness unacceptable behavior, or discover behavior that violates this Code of Conduct, please report it by opening a private discussion or contacting the project maintainer.

# ⚖ Enforcement

Project maintainers are responsible for clarifying and enforcing this Code of Conduct.

Maintainers may remove, edit, or reject comments, commits, pull requests, issues, or other contributions that do not align with this Code of Conduct.

Repeated or severe violations may result in temporary or permanent restrictions from participating in the project.

---

# ❤️ Our Goal

Our goal is to build an enjoyable, educational, and collaborative open-source cricket strategy simulator where developers of all skill levels can learn, contribute, and improve together.

Thank you for helping make **Invincible – IPL Strategy Simulator** a welcoming project for everyone.
