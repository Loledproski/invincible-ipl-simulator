# 🏏 Invincible - IPL Strategy Engine

**Invincible** is an offline, tactical cricket draft simulator built with React, Tailwind CSS, and Framer Motion. Step into the shoes of a franchise manager, draft a perfect 11-player squad from randomized historical IPL rosters, and attempt to sweep a grueling tournament campaign without losing a single match.

![Invincible IPL Simulator](https://i.imgur.com/your-image-placeholder.png) *(Replace with a screenshot of your app)*

## 🌟 Features

* **Draft Your Dynasty**: Pick your favorite IPL franchise (Mumbai Indians, Chennai Super Kings, Royal Challengers Bangalore, etc.) and adapt the app's visual theme to match your team's colors.
* **Historical Player Database**: Draft legendary and modern players from historical IPL seasons, each with unique roles and OVR (Overall) ratings.
* **Tactical Balance Engine**: The engine actively monitors your squad composition. Fail to pick a wicketkeeper or enough bowling options, and your team's overall rating will suffer massive penalties.
* **Lineup & Captaincy**: Organize your batting order and appoint a captain. Draft loyalists (10+ years at the same franchise) for chemistry bonuses.
* **Unforgiving Simulation**: Face off against historically accurate opponent XIs in a dynamic match simulation. A single defeat breaks your win streak and ends the campaign.
* **Share Your Glory**: Share your final run stats and campaign results directly to your clipboard.

## 🚀 Tech Stack

* **Frontend**: React 18
* **Build Tool**: Vite
* **Styling**: Tailwind CSS
* **Icons**: Lucide React
* **Animations**: Framer Motion
* **Deployment**: Google Cloud Run (via Google AI Studio)

## 🛠️ Local Development

To run this project locally on your machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/invincible-ipl-simulator.git
   cd invincible-ipl-simulator
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
